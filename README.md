# 🎨 BUTUSIC – Creative Landing Page

A visually appealing and modern landing page built using **HTML5** and **CSS3**. The project showcases an artistic exhibition concept with bold typography, creative layouts, editorial-inspired design, and custom illustrations.

## ✨ Features

- Modern editorial-style landing page
- Responsive layout
- Pure HTML & CSS (No JavaScript)
- Clean and organized code structure
- Custom illustrations and graphics
- Remix Icons integration
- Creative UI inspired by digital exhibitions

## 🛠️ Built With

- HTML5
- CSS3
- Remix Icons
